🔧 故障排除

如果主页显示异常，请尝试：

  1. 强制刷新: Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)
  2. 清除缓存: 在浏览器开发者工具中清除缓存
  3. 检查控制台: 按F12查看是否有JavaScript错误
  4. 重启服务器: 在终端中按Ctrl+C停止，然后运行 npm run dev
